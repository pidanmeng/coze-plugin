# 欢迎使用 Markdown 海报生成器

这是一个功能强大的工具，可以将 Markdown 内容转换成精美的海报！

## 主要特性

- **简单易用** - 只需要编写 Markdown
- **高度自定义** - 支持自定义尺寸和样式
- **快速生成** - 实时预览和下载

### 支持的语法

1. 标题（H1-H6）
2. 粗体文本：**加粗文字**
3. 斜体文本：*倾斜文字*
4. 删除线：~~删除文字~~
5. 行内代码：`console.log('Hello')`

> 这是一个引用块示例

---

#### 代码示例

```javascript
function greet(name) {
  return `Hello, ${name}!`;
}

console.log(greet('World'));
```
